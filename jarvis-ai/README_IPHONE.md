## 🎯 **Cara Pasang JARVIS AI di iPhone 12 Pro Max**

### 📋 **Prerequisites:**
- iPhone 12 Pro Max (iOS 15+)

## 🚀 **Method 1: Web App (Recommended)**

### **Step 1 - Setup di PC/Mac:**
```bash
1. Buka folder: c:\Users\cctv\jarvis-ai\
2. Pastikan semua files ada:
   - index.html
   - styles.css  
   - script.js
   - manifest.json
   - sw.js
```

### **Step 2 - Host ke iPhone:**

#### **Option A: Local Network (Easiest)**
```bash
# Install Python jika belum ada
# Buka terminal/command prompt di folder jarvis-ai
cd c:\Users\cctv\jarvis-ai

# Start local server
python -m http.server 8000

# Atau guna Node.js
npx http-server -p 8000
```

#### **Option B: File Transfer**
```bash
1. Copy folder jarvis-ai ke Google Drive/Dropbox
2. Share link ke iPhone
3. Download ke iPhone Files app
```

### **Step 3 - Install di iPhone:**

#### **Pergi ke URL di iPhone Safari:**
```
http://192.168.1.100:8000  (ganti dengan IP PC anda)
```

#### **Add to Home Screen:**
1. **Buka Safari** di iPhone
2. **Pergi ke URL** JARVIS AI
3. **Tap Share button** (📤 bottom bar)
4. **Scroll down** → "Add to Home Screen"
5. **Name:** "JARVIS AI"
6. **Tap "Add"**
7. **Done!** 🎉

## 🔧 **Troubleshooting:**

### **❌ Auto Mic Not Working:**
```bash
Solution:
1. Settings → Safari → Clear History & Website Data
2. Restart Safari
3. Re-add to Home Screen
4. Allow microphone permissions again
```

### **❌ Can't Hear Voice:**
```bash
Solution:
1. Check iPhone volume (not silent)
2. Settings → Sounds & Haptics → Toggle off "Change with Buttons"
3. Increase volume using side buttons
4. Check Bluetooth not connected
```

### **❌ Page Not Loading:**
```bash
Solution:
1. Check PC/Mac server is running
2. Verify same WiFi network
3. Try different browser (Chrome/Firefox)
4. Check firewall settings
```

### **❌ Microphone Permission:**
```bash
Solution:
1. Settings → Privacy & Security → Microphone
2. Find Safari/JARVIS AI
3. Toggle ON
4. Restart app
```
