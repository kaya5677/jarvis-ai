# JARVIS AI Assistant

Aplikasi JARVIS AI Assistant yang moden dan interaktif dengan ciri pengecaman suara dan text-to-speech. **Menyokong Bahasa Melayu Malaysia dan English sahaja.**

Modern and interactive JARVIS AI Assistant application with voice recognition and text-to-speech features. **Supports Malay (Malaysia) and English only.**

### **Step 2: Enable GitHub Pages**
```bash

1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: / (root)
5. Save
```

### **Step 3: Access from iPhone**
```bash

https://[your-username].github.io/jarvis-ai/

# Add to iPhone Home Screen:
1. Open URL in Safari
2. Tap Share → Add to Home Screen

```

## 🔧 **Troubleshooting:**

### **iPhone Issues:**
```bash
❌ Auto Mic not working:
→ Settings → Safari → Clear History
→ Restart Safari
→ Re-add to Home Screen

❌ No voice:
→ Check iPhone volume (not silent)
→ Settings → Sounds → Turn off "Change with Buttons"

❌ Cannot load:
→ Check internet connection
→ Verify GitHub URL is correct
→ Try different browser
```

### **Desktop Issues:**
```bash
❌ Voice recognition not working:
→ Use Chrome or Edge
→ Allow microphone permissions
→ Check microphone hardware

❌ Auto Mic not responding:
→ Refresh page
→ Check browser permissions
→ Update browser
```
